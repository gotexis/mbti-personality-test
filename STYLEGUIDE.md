# Style Guide

## 设计参考
- https://www.16personalities.com — 借鉴了人格类型卡片分组布局、渐变色头部、进度条测试 UX
- https://www.truity.com/test/type-finder-personality-test-new — 借鉴了7级量表交互方式、简洁的问题卡片布局
- https://personalitymax.com — 借鉴了结果展示的维度百分比条形图

## 配色方案
DaisyUI theme: nord
- Primary gradient: purple-600 → indigo-600 (header/hero)
- Group colors: purple (Analysts), green (Diplomats), blue (Sentinels), amber (Explorers)

## 设计决策
- Hero section 使用渐变背景突出 CTA
- 16类型按4组展示，每组有独特配色
- 测试页使用分页（每页10题）减少滚动疲劳
- 结果页使用进度条展示各维度百分比
- 类型详情页使用对比色卡片展示优缺点
